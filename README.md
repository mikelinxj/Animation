## Installation

Una vez dentro de la aplicacion Processing, para ejecutar la música, se debe de descargar:

```bash
    - Click en "Sketch"
    - Click en "Importar biblioteca..."
    - Click en "Gestionar bibliotecas..."
    - Buscar "Sound"
    - Intalar
    - Disfrutar
```
